# ☕ Coffee Shop Sales Analysis Dashboard

> **Retail sales analysis to uncover actionable insights that enhance Coffee Shop performance.**

---

## 📌 Project Overview

This project analyzes real-world retail sales data from a coffee shop chain using **Microsoft Excel**. The goal is to transform raw transactional data into a clean, interactive dashboard that answers key business questions — from peak hours to bestselling products.

---

## 🎯 Objectives

- Analyze how sales vary by **day of the week** and **hour of the day**
- Identify **peak times** for sales activity
- Track **total monthly revenue** trends
- Compare performance across **different store locations**
- Calculate the **average order value per person**
- Identify **bestselling products** by quantity and revenue
- Understand sales distribution by **product category and type**

## 🗃️ Dataset Details

The raw dataset contains transactional records from a coffee shop chain. Key columns include:

| Column | Description |
|---|---|
| `transaction_date` | Date of the transaction |
| `transaction_time` | Time of the transaction |
| `transaction_qty` | Quantity of items purchased |
| `store_location` | Store branch name |
| `product_category` | Category of the product |
| `product_type` | Type within the category |
| `product_detail` | Specific product name |
| `unit_price` | Price per unit |

---

## 📊 Dashboard Features

The Excel dashboard covers:

- 📅 **Sales by Day & Hour** — Heatmap-style view of when customers buy most
- 📈 **Monthly Revenue Trend** — Line/bar chart showing revenue growth over time
- 🏪 **Sales by Store Location** — Compare how each branch is performing
- 🧾 **Avg. Order Value** — Per-person spend analysis
- 🏆 **Top Products** — Best sellers by quantity and revenue
- 🗂️ **Category Breakdown** — Revenue split across product categories

## 🔍 Key Insights

- ⏰ **Peak hours** were identified in the morning (8 AM – 10 AM), typical of a coffee shop's busiest window
- 📆 **Weekdays** drive more consistent revenue compared to weekends
- 🏪 Certain **store locations** significantly outperform others in total revenue
- ☕ **Coffee** remains the dominant product category by both quantity and revenue
- 💵 The **average bill per person** offers a benchmark for upselling strategies
